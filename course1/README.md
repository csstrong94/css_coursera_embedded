to build all supporting files 
(object files, preproccessor outputs, assembly files, linker map, etc):

make build-all


---------------------------

to compile only:

make compile


---------------------------

to build without full suite of output files:

make build


---------------------------

to make specific file:

make <file output name>

i.e make memory.o


---------------------------

to clean:

make clean


---------------------------

to execute:

./main.out
